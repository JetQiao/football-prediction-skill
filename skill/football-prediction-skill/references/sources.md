# 数据源与降级

## 优先级

- 竞彩赛单：SportteryAPI REST/MCP → 中国体育彩票公开接口本地直连 → 用户提供快照。
- 实时市场：Pinnacle/付费聚合源 → 多家欧指共识 → 无市场层并明确降级。
- 实力特征：xG + ClubElo → 单独 Elo → 联赛中性先验。
- 伤病阵容：官方信息 + API-Football → 有来源媒体 → 缺失并降低置信度。
- 历史回测：football-data.co.uk；运行时下载，不随 Skill 分发。2025-07 后其 Pinnacle 数据可能滞后，优先使用市场平均收盘字段。

## 约束

- 缓存原始响应并记录时间，但不要提交第三方数据集。
- 控制请求频率，遵守来源条款和 robots 策略。
- SportteryAPI 的全球 Worker 可能因上游地域限制无法拉实时数据；优先本地运行。
- 只有同一预测截点可获得的数据才能进入回测。历史 Pinnacle 收盘赔率可作收盘基线，不得伪装成更早时点输入。
